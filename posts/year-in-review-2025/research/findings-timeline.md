## SQL Queries Used (For Methodology Section)

### Query 1: Monthly AI Article Count
```sql
SELECT
  strftime('%Y-%m', date) as month,
  COUNT(*) as ai_articles
FROM headlines
WHERE
  (title LIKE '%AI%' OR title LIKE '%artificial intelligence%')
  OR (summary LIKE '%AI guideline%' OR summary LIKE '%AI policy%'
      OR summary LIKE '%ChatGPT%' OR summary LIKE '%AI-generated%'
      OR summary LIKE '%AI hallucination%')
GROUP BY month
ORDER BY month;
```

### Query 2: Accountability Indicators
```sql
SELECT date, title, summary
FROM headlines
WHERE (summary LIKE '%fine%' OR summary LIKE '%discipline%'
       OR summary LIKE '%policy%' OR summary LIKE '%hallucination%'
       OR summary LIKE '%risk%' OR summary LIKE '%sanction%')
  AND (title LIKE '%AI%' OR summary LIKE '%AI%')
ORDER BY date;
```

### Query 3: Adoption Indicators (Pre-October)
```sql
SELECT date, title, summary
FROM headlines
WHERE (summary LIKE '%training%' OR summary LIKE '%efficiency%'
       OR summary LIKE '%opportunity%' OR summary LIKE '%adopt%'
       OR summary LIKE '%pilot%' OR summary LIKE '%innovation%'
       OR summary LIKE '%enhance%' OR summary LIKE '%improve%')
  AND (title LIKE '%AI%' OR summary LIKE '%AI%')
  AND date < '2025-10-01'
ORDER BY date;
```
