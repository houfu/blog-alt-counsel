## 2. Original Blog Post — Full Text

**Source:** https://www.alt-counsel.com/the-solo-counsel-reality-what-minlaws-ai-guidelines-miss-about-in-house-practice/

### Structure (for reference to avoid repetition):
1. **Opening** — Appreciation for the guide's enterprise coverage, but "as a solo in-house counsel, I'm struck by how disconnected it feels from my daily reality"
2. **Section 1: Professional Judgement Burden** — "professional judgement fallacy," Mata v. Avianca example, time/pressure/resource constraints, validation fatigue, need for AI literacy
3. **Section 2: Shadow AI** — Enterprise framework ignores consumer tool reality, regulatory gap, no audit trails
4. **Section 3: Transparency vs In-House Dynamics** — Different stakeholder relationships, education overhead, risk perception mismatch
5. **Section 4: What Solo Counsel Actually Need** — Practical risk assessment, shadow AI management, resource-conscious implementation, context-sensitive disclosure
6. **Closing** — "Maybe I'm not the target audience for this guide. But that itself is the problem."

## 4. Draft vs Final — Key Structural Differences

| Aspect | Draft (Sep 2025) | Final (Mar 2026) |
|---|---|---|
| Length | ~26 pages | ~48 pages (+annexes) |
| Scope | "lawyers, in-house counsel, paralegals and legal secretaries" | Expanded to include "allied legal professionals (legal technologists, legal project managers), alternative legal service providers, law students, and anyone providing GenAI tools" |
| AI Literacy | Not mentioned | New para 20(a) — 5-point AI literacy requirement |
| Consumer tools | Brief mention in 19(c) — 3 bare lines | Detailed para 23(d) with specific protocols, notes, and warnings |
| Bias | One sentence in para 16 | Expanded para 17 with 3 sources of bias + new para 20(e) on mitigation |
| Risk framework | None | New Diagram 2 — human-in-the-loop vs human-on-the-loop |
| SCCA reference | Not mentioned | SCCA Code of Ethics referenced throughout for in-house counsel |
| Firm examples | Inline examples (Clifford Chance, WongPartnership) | Expanded to Annexes A-B with 7+ firms including in-house (GenZero) |
| Templates | None | Annexes C-E: sample governance policy, employee handbook, engagement letter, vendor checklist |
| Adoption stages | Generic steps | 3-stage maturity model (Stage 1/2/3) mapped to each step |
| Junior lawyers | Not addressed | Para 45 on developing junior lawyers holistically |
| Supervision | Not addressed | Para 19 on supervisory duty including AI literacy |
| Cybersecurity | Brief mention | Detailed para 23(e)(v) + references to CSA resources |
