# Pre-Review Checklist (Run Before Sending to Reviewers)

## Audience Value
- [ ] Does opening paragraph answer "why should I care?"
- [ ] Are all examples concrete (not abstract)?
- [ ] Is there a clear takeaway in first 100 words?

## Common Gaps (From Your History)
- [ ] Security/compliance addressed? (If applicable)
- [ ] Learning curve honestly stated? (Not oversold)
- [ ] Regional context included? (Singapore/ASEAN angle)
- [ ] Economics/ROI made concrete? (Not abstract)
- [ ] Clear CTA with specific next step?

## Repetition Check
- [ ] Each section adds NEW information?
- [ ] No circular restatements of thesis?
- [ ] Examples used only once?

## Authenticity
- [ ] First-person claims are true to your experience?
- [ ] No false humility or false confidence?
- [ ] Vulnerable moments feel earned?
